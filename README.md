# 儿童《论语》学习网站 / こどものための論語

一个面向儿童的《论语》学习静态网站，支持完整 20 篇原文浏览、中文儿童导读、日语提示、浏览器朗读、搜索和学习进度记录。

## 功能

- 完整《论语》20 篇，当前底本共 501 条章句
- 每条章句自动生成儿童化主题导读和关键词
- 支持原文/篇名/主题搜索
- 支持繁体原文和简体辅助显示切换
- 支持按学习、孝亲、仁爱、礼仪、诚信等主题筛选
- 首页提供“今日一句”“继续学习”“随机一句”三个入口
- 支持日语朗读原文、中文朗读导读
- 支持标记“已记住”，进度保存在浏览器 localStorage
- 纯静态页面，直接打开 `index.html` 即可使用

## 说明

当前版本已经从 demo 数据扩展为完整原文版。全量日文训读、逐字 Ruby 注音和逐句人工翻译尚未逐条校订，后续可以在 `data.js` 中继续为每条章句补充更精细的日文读法与译注。
