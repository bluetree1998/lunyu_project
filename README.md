# 🎋 儿童论语学习网站 / こどものためのろんご

一个面向儿童的论语学习网站，支持日文 Ruby 注音、中文解释和音频朗读。

## ✨ 功能特色

- 📖 **日文 Ruby 注音** — 每句论语标注完整振り仮名
- 🈶 **中文双重解释** — 原文解析 + 儿童简单版
- 🔊 **音频朗读** — 日文和中文两种朗读（Web Speech API）
- ⭐ **记忆标记** — 标记已记住的句子，本地存储进度
- 📈 **学习进度条** — 实时显示学习完成百分比
- 🐼 **吉祥物互动** — 右下角熊猫给予鼓励

## 📚 内容篇目

| 篇 | 日文读音 | 句数 |
|---|---|---|
| 学而篇 | がくじへん | 3 句 |

| 里仁篇 | りじんへん | 2 句 |
| 述而篇 | じゅつじへん | 2 句 |
| 子罕篇 | しかんへん | 1 句 |

## 🚀 使用方法

直接用浏览器打开 `index.html` 即可，无需服务器。

## 🛠️ 技术栈

- HTML5 + CSS3 + JavaScript (Vanilla)
- Web Speech API（音频朗读）
- Google Fonts（Nunito）
- localStorage（进度保存）
